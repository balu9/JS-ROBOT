JavaScript Robot Simulator

The application can read commands in following format (not case sensitive),Press Enter after giving command to the ROBOT :)

  • PLACE X-COORDINATE, Y-COORDINATE, DIRECTION (Example: place 2, 3, west)
  • MOVE (To move the robot forward in the direction it is facing)
  • LEFT (Rotate left)
  • RIGHT (Rotate right)
  • REPORT (To see the current position of robot on the grid)
